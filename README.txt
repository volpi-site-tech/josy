VOLPI — Site multipágina (Premium)
=================================

Conteúdo:
- index.html
- sobre.html
- catalogo.html
- contato.html
- styles.css
- app.js
- logo.svg
- netlify.toml
- images/  (coloque aqui as imagens: destaque_home.jpg e foto_josy.jpg)

Como publicar:
- Netlify (drag & drop do conteúdo descompactado)
- GitHub + Netlify: crie repo e envie os arquivos (não envie .zip)

Observações:
- O catálogo funciona localmente usando localStorage. Para uso profissional, conectar a um CMS (Airtable, Firebase, Supabase).
- Formulário de contato usa mailto; pode-se integrar com Netlify Forms ou serviço de backend.
