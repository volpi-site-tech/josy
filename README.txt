
VOLPI - Site (final)
====================

Estrutura:
- index.html (homepage)
- sobre.html (about)
- catalogo.html (catalog)
- contato.html (contact)
- css/estilos.css
- js/app.js
- img/* (placeholders)

Como editar:
- Substitua as imagens em img/ pelos arquivos reais mantendo os nomes (imovel1.jpg, imovel2.jpg, imovel3.jpg, banner.jpg, josy.jpg).
- Os títulos/descrições dos cards são contenteditable na página (clique e edite).
- Para deploy no Netlify, conecte o repositório GitHub e aponte o diretório root (publicar a partir de /). Arquivo netlify.toml incluído para referência.
